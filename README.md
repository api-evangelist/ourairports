# OurAirports

OurAirports is an open airport data platform maintained by a global community of aviation enthusiasts. It provides a comprehensive, community-maintained database of 85,000+ airports, runways, radio navigation aids (navaids), and communication frequencies worldwide. All data is released to the public domain under the Unlicense license and is available as daily-updated CSV datasets hosted on GitHub Pages.

## Data Available

| Dataset | Description | Download |
|---|---|---|
| airports.csv | 85,000+ airports with codes, coordinates, type, and elevation | [Download](https://davidmegginson.github.io/ourairports-data/airports.csv) |
| runways.csv | Runway surfaces, dimensions, lighting, and endpoint coordinates | [Download](https://davidmegginson.github.io/ourairports-data/runways.csv) |
| navaids.csv | VOR, NDB, DME, TACAN, and VORTAC navigation aids | [Download](https://davidmegginson.github.io/ourairports-data/navaids.csv) |
| airport-frequencies.csv | TWR, GND, ATIS, UNICOM radio frequencies in MHz | [Download](https://davidmegginson.github.io/ourairports-data/airport-frequencies.csv) |
| countries.csv | ISO 3166-1 country reference data | [Download](https://davidmegginson.github.io/ourairports-data/countries.csv) |
| regions.csv | ISO 3166-2 administrative subdivision reference data | [Download](https://davidmegginson.github.io/ourairports-data/regions.csv) |
